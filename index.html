Jönköping City Clone

This project hosts a PostgreSQL server on docker. I write js files that communicate between the backend server to the frontend handling GET POST PUT and DELETE request as well as how the server should respond. I get SQL data and convert it into json. That data is the used to populate HTML elements created through Javascript and placed in the DOM. The user can update and add to the database through the frontend.
